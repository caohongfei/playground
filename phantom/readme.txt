1. 依赖环境：
	node.js v10.15.3 （这个版本是测试过了，小红的脚本需要的环境是14.17.1，虽然没有测试过，应该也可以）
	通过运行 node -v 可以检查已经安装的node版本

2. 代码文件 search.js ，需要修改以下值
	把第4行的 ROOT 后面引号里的值改成小红的程序输出结果的目录完整路径
	把第5行的 NAMES_FILE 后面引号里的值改成“离职人员检索关键词列表202107.txt”这个文件在本机的完整路径
	把第6行的 OUT_DIR 后面引号里的值改成希望的输出结果存放的目录的完整路径

3. 执行以下命令，大概需要几分钟，完成后即可在 OUT_DIR 目录下得到结果
	node search.js
